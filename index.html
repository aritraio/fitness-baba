⚠️ Open via localhost — not as a file
Browsers block API calls from file:// URLs (CORS null-origin). Run a local server in the same folder, then open the URL it gives you:

Python 3 python3 -m http.server 8080 copy
Node npx serve . copy
Recommended: use npx vercel dev — it runs the API proxy and auth locally too.
